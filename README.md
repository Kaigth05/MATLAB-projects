# MATLAB-projects
Diverse codes for linear algebra, numeric methods and differential equations
In this space, you can see three different folders:

- Firstly, we have some codes from Informatical Tools in which we work with equations and functions trying to approximate solutions and extreme points.
- Secondly, a folder called "2024-5" in which we have two folders form Numerical Methods for Linear Algebra and Ordinary Differential Equations.
- Finally, a folder called "2025-6" in which we have multiple folders dividing some practices and seminaries of two different subjects: Numerical Approximation and Mathematical Modelling.

You can access via online to the codes through the next link or download them directly from the main page of the repository: https://drive.mathworks.com/sharing/15a943f5-c249-4988-af31-f8fcc9aed770
